Necromancer Armor

Necromancer armor is a Hardmode, post-Mech armor set consisting of either the Necromancer Crown or Necromancer Mask, alongside the Necromancer Breastplate and Necromancer Greaves. In order to craft the full set the player will need 66 Chlorophyte Bars, 4 Dark Shards and 28 of any evil substance.

A full set of Necromancer armor provides the following benefits:

- +31 defense
- +25% summon damage
- +15% summon attack speed
- +1 minion slot
- +25% artifact damage
Set bonus (Crown)
- Slaying an enemy provides a stackable, temporary boost to all stats
- Mana usage is halved when in a Graveyard
- Artifact minion mana costs are halved
- Enemies spawn more frequently
- +3 minion slots
Set bonus (Mask)
- Summons an unsatisfied soul for every 2 minion slots you have from slain enemies
- Mana usage is halved when in a Graveyard
- Enemies spawn more frequently
- +3 minion slots
