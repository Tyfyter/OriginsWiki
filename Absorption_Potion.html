Absorption Potion

[statblock Absorption_Potion]

The Absorption Potion is a buff potion that can be crafted using Bottled Water, a Bomb Fish and a Fallen Star. When consumed, it provides the Safe From Yourself buff, which provides immunity to all self-damage and knockback the player inflicts to themselves from one of their own explosives. This effect will last for 6 minutes, but can be canceled anytime using the Interact button. Unlike most self-damage boosts, it does not affect any self-damage from other sources.
