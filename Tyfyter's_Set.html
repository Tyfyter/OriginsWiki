Tyfyter's set

Tyfyter's set are vanity developer items that can be obtained rarely (6.25%) from Treasure Bags dropped from Hardmode bosses (except Queen Slime's). It is the vanity set made for Tyfyter, the programmer behind Terraria Origins.

These items are meant to be equipped in the social slots as they only provide a visual change.

Tyfyter's equipment also includes non-vanity items, which will drop in addition to his clothing set:
