Automated Returns Handler

The Automated Returns Handler is a Hardmode, post-Plantera accessory crafted using Master Ninja Gear and a Super Turbo Reel. While equipped, the player has the ability to climb and slide down walls, dash by double-tapping the horizontal movement keys, a 10% chance to dodge any attack and an increased return speed for hooks and harpoons by 2.
