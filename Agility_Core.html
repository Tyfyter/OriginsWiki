Agility Core

The Agility Core is a post-Moon Lord, Eyndum-tier accessory that is crafted from materials obtained through the Test of the Ancients event at an Elderforge. It can only be equipped in a dedicated core slot provided by a full set of Eyndum Armor that appears to the left of the player's equipment section. It provides the following increases:

- +100% flight time
- +300% movement speed
- +200% acceleration/deceleration speed
- +200% running speed
- +5px/s jump speed
