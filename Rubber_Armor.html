Rubber Armor

Rubber armor is an armor set consisting of the Rubber Helmet, Rubber Breastplate and Rubber Greaves. In order to craft the full set the player will need 50 Rubber.

A full set of Rubber armor provides the following benefits:

- +7 defense
- +5% generic critical strike chance
- +10% damage reduction
- +5% generic damage
Set bonus
- Immunity to Static Shock and Electrified
- +5% damage reduction
