Fiberglass Armor

Fiberglass armor is an armor set consisting of the Fiberglass Helmet, Fiberglass Pauldrons and Fiberglass Boots. The Fiberglass Weaver drops any of the one pieces of the set or two at a time in Expert Mode. In order to obtain the full set the player will need to slay the Fiberglass Weaver at least 3 times.

A full set of FIberglass armor provides the following benefits:

- +16 defense
Set bonus
- +4 damage generic damage
