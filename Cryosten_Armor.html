Cryosten Armor

Cryosten armor is an armor set consisting of the Cryosten Helmet, Cryosten Breastplate and Cryosten Greaves. In order to craft the full set the player will need the full Snow armor set, 70 Ice Blocks and 3 Life Crystals.

A full set of Cryosten armor provides the following benefits:

- +12 defense
- +1 HP/s life regeneration
- +40% explosive blast radius
- +8% max max life
- +10% movement speed
- Increased movement speed on ice
Set bonus
- All attacks inflict Frostburn for 5-10 seconds
- +10% explosive velocity
- Immunity to Chilled and Frozen
