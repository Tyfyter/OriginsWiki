Configuration options

Origin's configuration options are changable settings provided by the mod in its in-game mod configuration menu. Configuration options can be set to modify visual or gameplay details as players see fit.

Accessibility options (client)

- Disable Defiled Wastelands Shader: Disables the noise and desaturation filter for when in the Defiled Wastelands.

Accessibility options (server)

- Riven Assimilation Torn multiplier: Reduces the Torn severity applied by Riven Assimilation.

General options

- Set bonus double-tap: Requires a specified key binding (Down button by default) to be used twice in a short time-frame for set bonus abilities to trigger.
- Animated Ravel transformation: Makes Ravel transformations flashier.
- Defiled Shader Jitter: Modifies the intensity of tiles and objects jittering (shaking) on a scale from 0-100% in 5% increments.
- Defiled Shader Noise: Modifies the intensity of the noise overlay on a scale from 0-100% in 5% increments.
- Extra-gooey Riven gores: Makes Riven enemies release a gooey gore that sticks to the player, NPCs, and other surfaces. Worm-like Riven enemies release a lot of these gores which may hinder performance.
- 24-hour time: Makes Origins' clocks display time in a 24-hour format.

Laser Tag HUD

- Health Pip Placement: Adjusts the location of the Laser Tag health bar to the bottom, left, right, top, back or front.
- Health Pip Offset: Offsets the health pips in increments of 0-32.
- Health Pip Direction Inverted: Inverts the direction health pips appear.

Journal options

- Auto-open on unlock: Opens the Journal when inspecting an item for the first time.
- Open Journal to: Changes the tab that the Journal always opens to (the Index, Quests, or Search tabs).

Vanilla Modifications

- Wooden item buffs: Whether or not vanilla/modded wooden items receive buffs from Origins.
- Royal Gel Slime: Allows Royal Gel to summon a slime minion.
- Volatile Gelatin boosts: Allows Volatile Gelatin's minion damage to count as explosive and benefit from bonuses.
- Rain armor buffs: Gives Rain armor pieces bonuses and a set bonus when equipped together.

Other

- Assimilation mechanics: Toggles between whether or not Assimilation will be active.
- Graveshield Zombie QOL: Makes Graveshield Zombies drop their tombstones as items instead of as tiles.
- Universal Grass Merge: Forces grass to merge with other types of grass.
