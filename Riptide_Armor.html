Riptide Armor

Riptide armor is an armor set consisting of the Riptide Helm, Riptide Breastplate and Riptide Greaves. In order to craft the full set the player will need a Diving Helmet, 39 Shell Piles, any Jellyfish and a Flipper.

A full set of Riptide armor provides the following benefits:

- +13 defense
- +10% magic
- +6 seconds of breath
- Emits a small aura of light
- +60 mana
- Grants the ability to swim
- +10% movement speed when submerged
Set bonus
- +5% magic damage when submerged or in rain
- Grants the ability to dash, releasing a damaging tidal wave projectile

Tips

- Riptide armor's dash can stop the player from receiving fall damage if used just before landing. It can additionally be used to temporarily maintain altitude.
