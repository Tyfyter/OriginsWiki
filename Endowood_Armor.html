Endowood Armor

Endowood armor is an armor set consisting of the Endowood Helmet, Endowood Breastplate and Endowood Greaves. In order to craft the full set the player will need 75 Endowood.

A full set of Endowood armor provides the following benefits:

- +4 defense
Set bonus
- +1 defense
