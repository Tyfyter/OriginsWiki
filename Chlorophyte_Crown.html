Chlorophyte Crown

The Chlorophyte Crown is a helmet armor piece to serve as the Summoner-equivalent for Chlorophyte armor. Equipping this as part of a full set does not change the set bonus beyond providing an additional 2 minion slots, otherwise only offering basic combat stats.

The Chlorophyte Crown provides the following benefits:

- +2 defense
- +12% summon damage
- +15% artifact damage
- +1 minion slot
Set bonus:
- +2 minion slots
