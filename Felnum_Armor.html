Felnum Armor

Felnum armor is an armor set consisting of the Felnum Helmet, Felnum Breastplate and Felnum Greaves. In order to craft the full set the player will need 60 Felnum Bars (180 Felnum Ore).

A full set of Felnum armor provides the following benefits:

- +16 defense
- +6% damage
- 10% movement speed
Set bonus
- Movement speed builds up the static charge of the armor which increases the damage dealt on the player's next strike
